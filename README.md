# Computação Gráfica - Híbrido

Repositório para entrega das atividades para cumprimento do Grau A de computação gráfica

## Entregas

Módulo 01 - Criação de repósitorio, adpatando o exemplo Hello3D, alterando o título da janela e as corres do triangulo

Módulo 02 - Intanciado 3 cubos na cena, substituindo a geometria do triangulo do exericio anterior para cubo, bem como adicionado controles de transação nos três eixos (X, Y, Z), de escala (aumentar ou diminuir), mantendo a rotação original em X, Y, Z.
